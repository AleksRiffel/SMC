# SMC Service

Сервис для генерации документов (РП, А, ФОС) на основе Excel файлов с учебными планами.

## Возможности

- Загрузка Excel файлов с учебными планами
- Извлечение информации о дисциплинах, компетенциях, объемах занятий
- Генерация документов по шаблонам:
  - Рабочая программа (РП)
  - Аннотация (А)
  - Фонд оценочных средств (ФОС)
- Скачивание сгенерированных документов
